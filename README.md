# LinearSineWave
Audio app written in .NET 9 and Avalonia UI

I am getting behind Avalonia UI and hope others follow. Coupled with .NET it is a fantastic platform to develop cross platform applications.

This application is still in the early stages of development!

Technologies Used:
* C#
* .NET 9
* Avalonia UI
* LibVlcSharp
* Community MVVM
* LiteDB
* Material Icons for Avalonia
